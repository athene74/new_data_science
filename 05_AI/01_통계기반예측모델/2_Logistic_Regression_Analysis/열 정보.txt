* State (주): 고객이 거주하는 미국의 주를 나타내며, 주의 약어로 표시됩니다 (예: KS는 캔자스주).
* Account Length (계정 유지 기간): 고객이 서비스 제공 업체와 함께한 기간을 나타내며, 일 단위로 표시됩니다.
* Area Code (지역 코드): 고객 전화번호의 지역 코드로, 일반적으로 고객의 대략적인 지리적 위치를 나타냅니다.
* Phone (전화번호): 고객의 전화번호 (익명화됨).
* Int'l Plan (국제 전화 요금제): 고객이 국제 전화 요금제를 사용하고 있는지 여부를 나타냅니다 (yes/no).
* VMail Plan (음성 메시지 요금제): 고객이 음성 메시지 요금제를 사용하고 있는지 여부를 나타냅니다 (yes/no).
* VMail Message (음성 메시지 수): 고객의 음성 메시지 수를 나타냅니다.
* Day Mins (주간 통화 시간): 고객이 주간에 사용한 총 통화 시간을 나타내며, 분 단위로 표시됩니다.
* Day Calls (주간 통화 횟수): 고객이 주간에 걸었던 통화 횟수를 나타냅니다.
* Day Charge (주간 통화 요금): 고객이 주간 통화에 대해 부과된 요금을 나타냅니다.
* Eve Mins (저녁 통화 시간): 고객이 저녁 시간에 사용한 총 통화 시간을 나타내며, 분 단위로 표시됩니다.
* Eve Calls (저녁 통화 횟수): 고객이 저녁 시간에 걸었던 통화 횟수를 나타냅니다.
* Eve Charge (저녁 통화 요금): 고객이 저녁 통화에 대해 부과된 요금을 나타냅니다.
* Night Mins (야간 통화 시간): 고객이 야간에 사용한 총 통화 시간을 나타내며, 분 단위로 표시됩니다.
* Night Calls (야간 통화 횟수): 고객이 야간에 걸었던 통화 횟수를 나타냅니다.
* Night Charge (야간 통화 요금): 고객이 야간 통화에 대해 부과된 요금을 나타냅니다.
* Intl Mins (국제 통화 시간): 고객이 국제 통화에 사용한 총 통화 시간을 나타내며, 분 단위로 표시됩니다.
* Intl Calls (국제 통화 횟수): 고객이 국제 통화를 한 횟수를 나타냅니다.
* Intl Charge (국제 통화 요금): 고객이 국제 통화에 대해 부과된 요금을 나타냅니다.
* CustServ Calls (고객 서비스 문의 횟수): 고객이 고객 서비스 센터에 문의한 횟수를 나타냅니다.
* Churn? (이탈 여부): 고객이 서비스에서 이탈했는지 여부를 나타내며, 'True'는 이탈, 'False'는 유지됨을 의미합니다.
